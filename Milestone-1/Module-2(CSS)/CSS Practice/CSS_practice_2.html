Learning CSS

Today I am learning css. the full meaning of css is cascading style sheet. There are 3 kinds of CSS.

  1. Inline CSS
  2. Internal CSS
  3. External CSS

In CSS there are many types of selectors.

  • element selectors
  • ID selectors
  • class selectors
  • universal selectors
  • combined selectors
  • attribute selectors
